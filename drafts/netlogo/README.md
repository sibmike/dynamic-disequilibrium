# NetLogo prototype (legacy)

`speculative_market_simulation.nlogo` is the original NetLogo implementation
of what the paper now calls the speculative-market EDEM variant. It was
previously the entire content of
[`sibmike/netlogo-realestate-simulations`](https://github.com/sibmike/netlogo-realestate-simulations)
(now superseded by `sibmike/dynamic-disequilibrium`).

The Python/Mesa port in [`../../python_simulation/`](../../python_simulation/)
supersedes this file for all current research; it is preserved here as the
reference NetLogo source the paper's [`§4`](../../paper/sections/04_implementation.tex)
implementation section translates from.

To run the original prototype, install [NetLogo 6.x](https://ccl.northwestern.edu/netlogo/)
and open `speculative_market_simulation.nlogo` directly.
